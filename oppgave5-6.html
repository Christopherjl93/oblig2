Git og GitHub

Git er et dataprogram som gjør det enklere å finne filer og endringer man har gjort på en oppgave ofte sammen med andre personer eller alene. I dataprogramet kan man søke etter ting man gjorde for noen måneder siden eller år tilbake, så lenge man hele tiden oppdaterer og legger inn endringer i Git.

GitHub er et web basert versjon av Git, som driver med hosting tjenester. GitHub tilbyr alle de samme tjenestene som Git, i tillegg til å legge til egne funksjoner. GitHub gir tilgangskontroll, samarbeidsfunksjoner som feilsporing, funkjsonsforespørsler og oppgavebehandling.

Fordelen med å bruke Git og GitHub er at man har en måte å dele kode, bilder eller endringer man har gjort på et prosjekt. Samtidig som at man har oversikt over tidligere versjoner av filene, så man kan gå tilbake å se endringene man har gjort.
